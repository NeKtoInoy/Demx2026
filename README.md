### Решение региона для ленивых!

Для решения 1 модуля я обьединил все команды в 1 скрипт deploy.sh. 
# Как использовать?!
1. Просто открываете файл deploy.sh и копируете содержимое
2. Любым способом вставляете его в терминал подключившись к виртуальной машине и жмете Enter
3. После чего пишем: ./deploy.sh ИМЯ_МАШИНЫ (Например ./deploy.sh ISP)
4. Скрипт сам запустит все команды для настроек, чтобы не копипастить по 200 раз.

# Такое надо проделать на каждой машине. 
## ВАЖНО! 
# Некоторые команды в скрипте зависят от интернета, поэтому начинать запускать скрипт лучше с ISP, потом HQ-RTR и BR-RTR. А остальные уже как хотите.
