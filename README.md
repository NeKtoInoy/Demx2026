# Модуль 1: Автоматизация развертывания

Решение для ленивых. Один скрипт для полной настройки всех машин регионального этапа. Больше никакого копипаста по 200 раз.

## Инструкция

Для настройки машин выполните следующие шаги:
1. Откройте `deploy.sh` и скопируйте всё содержимое.
2. На целевой BМ создайте файл и вставьте код (или просто вставьте в терминал).
3. Запустите: 
   ```bash
   ./deploy.sh ИМЯ_МАШИНЫ
   ```
   Вместо ИМЯ_МАШИНЫ пишем название той что хотим настроить например:
    ```bash
   ./deploy.sh ISP
   ```
# Порядок запуска (ВАЖНО!)
Так как скрипту требуется интернет для загрузки пакетов для команд, соблюдайте строгую последовательность:

1. `ISP` — настраиваем первым, чтобы поднять внешку
2. `HQ-RTR` — настраиваем вторым
3. `BR-RTR` — настраиваем третьим
4. Остальные ВМ — в любом порядке
